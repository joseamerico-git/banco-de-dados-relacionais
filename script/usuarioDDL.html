    Painel de navegação

    Personalizar a aparência do painel de navegação.

    No painel de navegação, substitui a árvore da base de dados por um seletor
    Liga com o painel principal destacando a base de dados ou tabela actual.
    Mostrar o logótipo no painel de navegação.
    URL para onde irá apontar o logótipo do painel de navegação.
    Abre a página ligada na janela principal (principal) ou numa nova janela (nova).
    Destaque servidor sob o cursor do rato.
    A quantidade de itens que podem ser mostrados em cada página no primeiro nível da árvore de navegação.
    Define a quantidade mínima de itens (tabelas, vistas, rotinas e eventos) a apresentar numa caixa de filtragem.
    Número máximo de tabelas utilizadas recentemente, definir 0 para desativar.
    Número máximo de tabelas favoritas, definir 0 para desativar.
    Configurar como 0 para colapsar o painel de navegação.
